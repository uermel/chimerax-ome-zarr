ChimeraX OME-Zarr plugin

OME-Zarr is a next generation file format specification for storing bioimaging data on the cloud.

This plugin provides limited support for OME-Zarr files in ChimeraX.

Currently supported:

  • ZYX-order multiscale volumes.
  • Opening local or remote zarr files.
  • Loading specific multiscales as volumes (streamed on demand and cached in memory).
  • Loading all multiscales as a single Volume, accessible using Chimerax's step setting.

Currently not (yet) supported:

  • Labels data.
  • Channel axes.
  • Time axes.
  • non-integer scaling.
  • translation transformations.
  • 2D data.
  • multi-image files.
